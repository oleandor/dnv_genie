Command: WRMorison

Description:
Define Morison in Analysis. You may define if velocity decomposition shall be used when calculating
the normal components of the drag forces, i.e. how to handle the v2 term in the Morison equation.
Example of use:
Add Morison to analysis
Analysis1.step(2).morison().normalToElement(true);
This command is scripted. Please read the user manual GeniE Vol. 6 for more information about Morison.
