Command: ExportMeshFem

Description:
Export mesh to FEM file
Example of use:
Export a finite element model (FEM) with a user given name
FemExporter = ExportMeshFem();
FemExporter.DoExport("C:/Location/T1.FEM");
This command is not scripted. You can include the export capability by editing
the journal file to typically include the code above
To include FEM export in a journal file, see the last Chapter of User Manual Volume 6
