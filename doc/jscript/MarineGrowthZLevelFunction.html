Command: MarineGrowthZLevelFunction

Description:
Define marine growth thickness, roughness height and density factor values as a function of vertical position.
Example of use:
Define a linearly varying value for marine growth
MyMarineVarGrowth = MarineGrowthZLevelFunction();
MyMarineVarGrowth.add (0m,0.2m,0.03m,2);
MyMarineVarGrowth.add (-50m,0.1m,0.02m,1);
MyMarineVarGrowth.useInForceCalculations(false);
This command is scripted
Define a linearly varying value for marine growth with elevations relative to seabed Z
MyMarineVarGrowth = MarineGrowthZLevelFunction();
MyMarineVarGrowth.add (50m,0.2m,0.03m,2);
MyMarineVarGrowth.add ( 0m,0.1m,0.02m,1);
MyMarineVarGrowth.useInForceCalculations(false);
MyMarineVarGrowth.setAbsoluteElevations(true);
This command is scripted
