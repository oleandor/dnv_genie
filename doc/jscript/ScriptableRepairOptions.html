Command: ScriptableRepairOptions

Description:
Repair options.
 
Property Summary
 bool RemoveCollapsedElements read/write
Collapsed elements will get removed when this option is checked.
 bool RemoveDisconnectedElements read/write
Not connected elements and their nodes will be removed. Nodes that are connected to other elements will not be removed.
 bool RemoveDisconnectedNodes read/write
Not connected nodes will get removed when this option is checked.
 bool SplitLargeAngleQuads read/write
Large angle quads Quad elements with an angle larger than the Corner angle >screening criteria will be split by the diagonal in the corner with the largest angle.
 bool SplitWarpedQuads read/write
Warped quads Quad elements with warping relative distance larger than the screening criteria are split.
