Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: ImportSINFileOptions

 
Property Summary
 ImportSINFileOptions createLoadCases read only
The 'Create load cases' option will create load cases for SIN result cases numbers with no matching FEM load case numbers, e.g. Analysis1.loadCase(N).
 ImportSINFileOptions matchLoadAndResultCase read only
The 'Match load and result cases' option will match the SIN result case numbers with the FEM load case numbers, when these exist.
 ImportSINFileOptions resultCaseAsSeparateEntities read only
The 'Separate result cases' option will let all result cases appear as separate entities, e.g. Analysis1.resultCase(N).
