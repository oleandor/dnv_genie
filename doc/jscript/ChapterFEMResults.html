Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: ChapterFEMResults

Description:
Document FEM Results. For more information, please read to GeniE Volume 6
Example of use:
ChapterFEMResults
demo_report = Report("demo_report");
demo_report.add(ChapterFEMResults());
demo_report.element(1).setCasePairs(Analysis1, Array(LC1_eqpm));
demo_report.element(1).setLoopLoads(true);
demo_report.element(1).setLoopSets(true);
demo_report.element(1).setSets(Array(bottom_deck));
demo_report.element(1).add(TableFEMBeamForceEnvelope());
demo_report.element(1).add(TableFEMBeamStressEnvelope());
demo_report.element(1).add(TableFEMNodeReaction());
demo_report.element(1).add(TableFEMNodeDisplacement());
demo_report.element(1).add(TableFEMBeamForce());
demo_report.element(1).add(TableFEMBeamStress());
demo_report.saveAs("demo_report.xml", mrWordXML);
a1 = Report("a");
a1.add(ChapterMasses());
a1.element(1).add(TableTotalMassAndCOG());
a1.element(1).add(TableSetMassAndCOG());
a1.element(1).add(TableSetContents());
a1.element(1).add(TableSetBoundingBox());
This command is scripted. Typically when documenting the "Beam Force Envelope",
the "FEM Beam Stress Envelope", the "FEMNodeReaction", the "FEM Node Displacement",
the "FEM Beam Force" and the "FEM Beam Stress" for Analysis1
This command is scripted. Typical parts of it may be as the example above.
Notice
Notice that the reaction forces will be printed for all finite nodes with a boundary condition
with reference to the finite element number. In case you specify a joint for a boundary condition,
the joint name is also used as a reference.
Finite element node displacements are listed for specified joints only
