Command: MassDensityFactor

Description:
Use this property to scale the mass of structural concepts without changing the materials.
Example of use:
Scaling the mass of a set
MassFactor1 = MassDensityFactor(2.6);
bottom_deck.massDensityFactor = MassFactor1;
This command is scripted, typically when scaling the mass of the set "bottom_deck" with a factor of 2.6)
It is possible to scale the masses to match a target mass from the context sensitive menu in the
Browser Utilities Sets. This option will use mass density factors to scale the masses.
I.e. no new material properties will be made.
Create Scaled Materials
St48_2_6 = St48.copy();
St48_2_6.density = St48.density * 2.6;
Bm1.material = St48_2_6;
Bm2.material = St48_2_6;
Bm3.material = St48_2_6;
This command is scripted, typically when making new materials affected of the mass scaling factor 2.6 (in this
case the material St48 is affected - a new material St48_2_6 is made and applied to the structural parts of the set).
Scale Beam Mass to represent a distributed load
var MDF = new Array();
      //Array of mass density factors
var count = 1;
for(var bm in ModelObjects) {
      //If you want to scale only a set:
      //for(var bm in MySet) {
if (bm.supportsType(TypeStraightBeam)) {
      // Find total UDL = UDLFactor*length
UDLFactor = 200.0 Kg; // 200 Kg/m
UDL = bm.length()* UDLFactor;
      // Find Mass density factor MDF = Target mass / Current mass
MDF[count] = MassDensityFactor((bm.mass().toDouble() + UDL.toDouble())/(bm.mass.toDouble()));
      // Make a set for the beam and scale
newSet=Set();
newSet.add(bm);
newSet.massDensityFactor = MDF[count];
Delete(newSet);
Print(bm.name + " Mass = " + bm.mass.toString());
count++;
}
}
If you need to run this several times for different sets,
only include "var count = 1;" the first time
