Command: ImportMeshFem

Description:
Import Conceptual information from Fem
Example of use:
Import geometry and loads from FEM file and create concept model
FemImporter = ImportMeshFem();
FemImporter.DoImport("C:/Location/T1.FEM");
This command is scripted and a typical example is given above
