Command: PrescribedDisplacement

Description:
An abstract Prescribed Displacement
Example of use:
Specify a prescribed displacement at a support point
Sp3.boundary = BoundaryCondition(Prescribed, Prescribed, Fixed, Fixed, Fixed, Fixed);
PDisp1 = PrescribedDisplacement(LC1, Sp3, 2mm, 5mm, 0 m, 0 deg, 0 deg, 0 deg);
This command is scripted. Typically for a support point that has prescribed
conditions in x and y direction with prescribed displacements of 2mm and 5mm respectively
It must be used in connection with a prescribed boundary condition.
