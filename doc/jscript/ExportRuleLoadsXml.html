Command: ExportRuleLoadsXml

Description:
Export Rule Load information (compartments) as XML file
for use by Nauticus Hull to create CSR rule loads.
Example of use:
Export a SIN results file with a user given name
RuleLoadsExporter = ExportRuleLoadsXml();
RuleLoadsExporter.DoExport("C:/Location/ yourname.xml");
This command is not scripted unless you tick off for such option
To include Rule Loads XML export in a journal file, read the User Manual Volume 6.
