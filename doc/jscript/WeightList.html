Command: WeightList

Description:
Equipment Weight list
Example of use:
Import Weight List
WeightList("C:/Location/Your_weight_list.xml");
Or
WeightList("C:Location/Your_weight_list.csv");
This command is scripted. Note that the above will import data, but not place the equipment
items in individual load cases, see Section 3.10 of User Manual Volume 1 for instructions on how to do this.
