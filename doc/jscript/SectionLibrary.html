Command: SectionLibrary

Description:
It is possible to import section profiles from section libraries.
Example of use:
Import Libraries or sections
OD101_6X10 = PipeSection(101.6mm,10mm);
OD101_6X10.description = "NVS lib : 101,6x10 NS-EN 10210/10219";
This command is scripted. The scripting depends on how many profiles that are selected.
A typical example is given above where one pipe section OD101_6X10 has been selected.
Import section profile
HE200A = ISection(190mm,200mm,6.5mm,10mm);
HE200A.description = "NVS lib : HE 200 A NS-EN 10034";
This command is scripted. The scripting depends on which section property that is imported.
Typically for a HE200A section from the NSF_EN library
