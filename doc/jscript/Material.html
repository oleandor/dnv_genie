Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: Material

Description:
Create and edit a material property for Linear or Shear Materials
Example of use:
Linear or Shear Materials
- For defining a New Linear Material, go to:
          MaterialLinear

- For defining a Shear Material, go to:
          MaterialShear

- Notice that it is also possible to import a material library from
          C:/Program Files/DNVS/GeniE/Libraries/Material_library.xml.
          The path name assumes you have installed the program using the default installation set-up.
          The library is imported by using the command File|Import|XML Concept Model.
          The library contains 71 linear isotropic materials; these are documented in User Manual Volume 3 Appendix B.
