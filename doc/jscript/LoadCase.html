Command: LoadCase

Description:
Define a Load Case
Example of use:
Define a load case where equipments, weight lists, and explicit loads are applied.
LC1 = LoadCase();
This command is scripted.
Define the load cases to be part of a code check run
1.
Cc1.run(1).addLoadCase(LC_total);

2.
Cc1.run(1).removeLoadCase(LC_total);
1. This command is scripted. Typically when using the load case "LC_Total" only
2. When removing a load case from the run this is the typical script command
