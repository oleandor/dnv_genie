DetectDisconnectedInnerBeams

bool DetectDisconnectedInnerBeams()

Description:
Automatically detect overlapping beams with double nodes. Identify which beams are the inner beams and assign a BeamTypeInnerBeamDisconnected property to these. These are the rules being used: Only consider overlapping nodes that has overlapping beam elements. Only consider beam elements that are C1 connected to the beam elements identified above. Accept (as the end of a beam) a single node that contains two overlapping beam elements. Disregard beam elements which has more than 2 incoming beam elements or where the 2 incoming beams are not C1.
