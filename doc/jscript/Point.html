Command: Point

Description:
Represent a geometric point
Example of use:
Insert a guide point and specify coordinates manually
Point1 = Point(7.5 m,5 m,0 m);
This command is scripted.
Insert a guide point graphically by snapping to 1 point
Point3 = Point(2.5 m,0 m,10 m);
This command is scripted.
