Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: ImportSacsPSI

Description:
Import a SACS PSI input file. The stand-alone application SacsPSI generates
the piles and soil data needed by GeniE, storing them in SacsPSIToGeniE.js.
Example of use:
Import a SACS PSI input file
GeniE restores the information from the SACS PSI input file by reproducing
the GeniE js-commands of relevant data.
