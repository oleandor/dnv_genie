Command: Beam

Description:
Define a basic beam and modify beam specific parameters like local system,
offset vector, split points, move ends, translate and buckling factors
Example of use:
Modify the local system of a beam
Bm21.rotateLocalX(90);
This command is scripted. Typically when rotating the beam Bm21 90 degrees around the beam local x-axis
Add eccentricities to the beam ends (Offset vector)
Bm21.setBeamOffset(Vector3d(0 m,0 m,-5cm));
This command is scripted. Typically when specifying an eccentricity of 5cm in negative global z-direction for beam Bm21
Add additional vertices (or snap points) to a beam
Bm31.splitAt(0.2);
Bm31.splitAt(0.4);
Bm31.splitAt(0.6);
Bm31.splitAt(0.8);
This command is scripted. Typically when distributing 4 points evenly of Bm31
Move one beam end
Bm31.extendEnd(2, -150 cm);

Bm31.moveEnd(1, Vector3d(0 m,5 m,0 m));
This command is scripted. Typically when reducing the length with 150 cm relative to the end point 2 of Bm31
The end point can be moved to another point, by a vector definition or extended/reduced
along its own local x-axis. In the latter case a positive value means extension of the beam
while a negative value means a reduction of the length.
Using a function to update the beam ends
var Bea = new Array();
Bea[1] = Dx1A1;
Bea[2] = Dx1C1;
Bea[3] = Dx1E1;
Bea[4] = Dx1G1;
End1_X_center = 0.15 m;
End2_X_center = 0.16 m;
End1_Offset = 1.837 m;
End2_Offset = 2.102 m;
End2_X_center = 0.16 m;
Main_Section = D39;
End1_Section = Node_Diagonal_X;
End2_Section = Node_Diagonal_X;
DiagonalProp (Bea,End1_Offset,End2_Offset,Main_Section,End1_X_center,End2_X_center,End1_Section,End2_Section);
First list the beams to be updated in an array
Define eccentricities and offsets for these beams
Then call a user function to do the update
DiagonalProp - function to update beams
function DiagonalProp (Bea,End1_Offset,End2_Offset,Main_Section,End1_X_center,End2_X_center, End1_Section,End2_Section)
{
    var NumBeam = Bea.length;
    for (i = 1; i < NumBeam ; i++) {
          Bea[i].section = Main_Section;
          if ( End1_X_center != 0. m ) {
              Bea[i].setEndOffset(1, Vector3d(End1_X_center, 0 m,0 m));
          }
          if ( End2_X_center != 0. m ) {
              Bea[i].setEndOffset(2, Vector3d(End2_X_center, 0 m,0 m));
          }
          if ( End1_Offset != 0. m) {
              Bea[i].divideSegmentAt (1,(End1_Offset / Bea[i].length()));
              Bea[i].SetSegmentSection(1, End1_Section);
          }
          if ( End2_Offset != 0. m) {
              Bea[i].divideSegmentAt(2,((Bea[i].getSegmentLength(2)-End2_Offset)/Bea[i].getSegmentLength(2)));
              Bea[i].SetSegmentSection(3, End2_Section);
          }
    }
Define eccentricities modelled by EndOffset
Then set End1 Offset and/or End2 offset
The beam is divided at relative position Lsegment / Lbeam
Move a beam with a translation vector (Translate)
Bm31.extendEnd(2, -150 cm);
Bm21.moveTranslate(Vector3d(0m,0m,2000mm));
This command is scripted. Typically when moving Bm21 2000mm in positive global z-direction)
Copy a beam using a translation vector
Bm2 = Bm1.copyTranslate(Vector3d(0 m,0 m,10 m));
This command is scripted. Typically copying Bm1 with the shown vector
Copy a beam by defining a rotation point, a vector and a rotation angle.
Bm103 = Bm21.copyRotate(Point(0 m,0 m,0 m), Vector3d(0,0,1),90 deg);
This command is scripted. Typically copying Bm1 around point (0,0,0) with vector (0,0,1) and angle 90 degrees
Copy a beam by defining a mirror plane built up from a point and a normal vector
Bm2 = Bm1.copyMirror(Point(0 m,5 m,0 m), Vector3d(0,1,0));

autoMSet = Set();
autoMSet.clear();
autoMSet.add(Bm1);
autoMSet.moveMirror(Point(0 m,5 m,0 m), Vector3d(0,1,0),geUNCONNECTED);
Delete(autoMSet);
This command is scripted. Typically copying Bm1 around point (0,5,0) with vector (0,1,0)
A typical move operation would be as shown above
Copy a beam by defining a scaling factor and a point used as the scaling centre
Beam1_1 = Beam1.copyScale(5,Point(0,0,0));
Beam2_1 = Beam2.copyScale(5,Point(0,0,0));
This command is scripted.
Copy beams by defining a source plane and a destination plane
Bm11 = Bm1.copy3Point(Point(27 m,0 m,10 m),Point(27 m,0 m,4 m),Point(27 m,28.5 m,10 m),
Point(37 m,0 m,10 m),Point(37 m,0 m,4 m),Point(37 m,28.5 m,10 m));

Bm12 = Bm2.copy3Point(Point(27 m,0 m,10 m),Point(27 m,0 m,4 m),Point(27 m,28.5 m,10 m),
Point(37 m,0 m,10 m),Point(37 m,0 m,4 m),Point(37 m,28.5 m,10 m));
This command is scripted. Typically copying Bm1and Bm2 using the input parameters above.
For both planes three points must be defined.
Copy a beam by defining a general transformation
This command is scripted depending on which operation is carried out.
For examples, see Section 3.3.12 of User Manual Volume 3.
Join beams and segments
Bm144.joinBeams(Bm145);
Bm144.joinSegments(1,2);
This command is scripted. Typically when joining Bm144 and Bm145
Divide beams and segments using a number of division criteria
Bm2 = Bm1.divide(XPlane3d(24 m));
This command is scripted. Typically when dividing Bm1 with an X-plane at x= 24 m (there are now two beams Bm1 and Bm2)
Delete the selection
Delete(Bm1);
Delete(Bm2);
This command is scripted. Typically when selecting Bm1 or Bm2
Add or modify properties to the selected beam
Bm1.section = HE600A;
This commands are scripted. Typically when adding profile type HE600A to Bm1
