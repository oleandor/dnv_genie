Command: SupportRigidLink

Description:
Represent a Support Rigid Link
Example of use:
Insert boundary conditions so that they are dependent upon a point.
Sp6 = SupportRigidLink(Point7, FootprintBox(Point8, Vector3d(0.2,0.6,1), LocalSystem(Vector3d(1 m,0 m,0 m), Vector3d(0 m,0 m,1 m))));
Sp6.includeAllEdges = true;
Sp6.rotationDependent = true;
Sp6.boundary = BoundaryCondition(Fixed, Fixed, Fixed, Fixed, Fixed, Fixed);
This command is scripted. Typically, when defining a rigid link Sp1 where the input parameters are documented above.
Independent point is Point7
- Box centre is Point8
- Box extent is dx = 0.2m, dy = 0.6m and dz = 1m
- Include all edges in region
- Rotations in dependent nodes same as for independent node
- Boundary condition: all degrees of freedom in independent point are Fixed
