Command: ExportModelJS

Description:
Create and export a clean journal file for frame and plate structures
Example of use:
Export a SIN results file with a user given name
JsExporter = ExportModelJS();
JsExporter.DoExport("C:/Location/yourname.js");
This command is not scripted. You can include the export capability by editing
the journal file to typically include the code above
To include JS export in a journal file, see the last Chapter of User Manual Volume 6
