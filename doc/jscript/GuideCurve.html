Command: GuideCurve

Description:
Define a Abstract guiding curve
Example of use:
Define curvature by using Fillet Curves
Curve4 = filletCurves(Curve2, Point(3 m,0 m,0 m), Curve3, Point(5 m,1 m,0 m), 2);
This command is scripted. Typically when selecting two curves and specifying a radius of 2 m.
The coordinate values are automatically proposed by GeniE
Create a fillet curve by clicking on two adjacent lines
Curve29 = filletCurves(Curve10, Point(5.688202143 m,5.688202143 m,0 m), Curve4, Point(5 m,4.148269176 m,0 m), 1);
This command is scripted. Typically when using a radius of 1 m
Insert a model curve by snapping between two points
Curve25 = ModelCurve(Array(Point(5 m,2.5 m,10 m),Point(6.327271284 m,2.174330704 m,14.9859724 m),Point(7.5 m,2.5 m,20 m)),3);
This command is scripted.
Getting the length of the GuideCurve
function GetLengthOfCurve(myCurve) {
curveLength = Beam(myCurve).length;
return curveLength;
}
exampleCurve = GuideSpline(Array(Point(5 m,0 m,0 m),Point(5 m,2.5 m,0 m),Point(2.5 m,5 m,0 m),Point(5 m,7.5 m,5 m),Point(7.5 m,7.5 m,5 m)),3);
print(GetLengthOfCurve(exampleCurve));
A beam based on the curve is created and the length of the beam (being the same as the length of the curve) is returned
