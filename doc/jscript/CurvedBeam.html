Command: CurvedBeam

Description:
A Curved Beam has the following limitation: Given the guide vector
(Beam.end2-Beam.end1), The angle between the guide vector and the tangen of the curved beam can never exceed 90 deg at any point along the beam
Example of use:
Define a Curved Beam
Bm1 = CurvedBeam(Point1,Point2,Point3);
Bm1 = CurvedBeam(Point(10 m,10 m,10 m),Point(10 m,20 m,20 m),Point(10 m,10 m,30 m));
Bm2=CurvedBeam(Point(0,0,0),Point(0,5,5),Point(0,10,0),Point(0,0,1)
Code for Bm1 gives a Legal curvature
Code for Bm2 gives a Illegal curvature
