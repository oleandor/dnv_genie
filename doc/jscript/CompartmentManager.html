Command: CompartmentManager

Description:
Automatically create compartments for use when creating loads from content or to define a tank subjected for filling
Example of use:
Define a CompartmentManager
My_Comp_Manager = CompartmentManager();
This command is scripted.
Give new name for all Compartment Managers
Cm = CompartmentManager();
var i = 1;
for(var comp in Cm.compartments){
var CompName = "MyComp" + (i + 1);
Print(comp.name());
i = i + 1; }
This command is scripted. For all in compartment manager,
give new name "MyComp1", "MyComp2", etc.
