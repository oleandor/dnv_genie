Command: PointMassMatrix

Description:
General point mass expressed as a mass matrix
Example of use:
Define a mass with different contribution in global translation and rotation x, y and z directions.
Mass2 = PointMassMatrix(Point(25,25,25),
5000 Kg, 4000 Kg, 3000 Kg, 10000 Kg*m^2,
8000 Kg*m^2, 6000 Kg*m^2);
This command is scripted. Typically, when adding different mass elements
using directional and a diagonal mass matrix
