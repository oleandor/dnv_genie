Command: StraightBeam

Description:
The Beam command creates a straight beam between two points
Example of use:
Define a Straight Beam
b1 = Beam(Point(0,0,0),Point(1,0,0));
b2 = Beam(b1.end1),Point(2,2,0));
This command is scripted.
Define a Straight Overlapping Beam
Bm4 = Beam(Point1,Point2,geAllowOverlap);
This command is scripted. Typically, when inserting Bm4 on top of another beam (automatically name) between
Point1 and Point2 when using reference point modelling.
Counting beams (or other items of interest
var counter = 0;
for (var object in ModelObjects){
if(object.supportsType(typeStraightbeam)){
counter = counter + 1
}}
print("Num of beams: " + counter)
This script is counting the number of straight beams in your model.
You could easily change it to counting other items of interest
by exchanging typeStraightBeam with typeFlatPlate, typeCurvedBeam, typeCurvedShell etc
You could easily change it to counting other items of interest
by exchanging typeStraightBeam with typeFlatPlate, typeCurvedBeam, typeCurvedShell etc
Additional Information
The beam can be
assigned a local coordinate system, and an offset (either constant beam offset or end offsets).
The beam can also have material, mesh density, cross section, element type,
and load interface properties assigned.
