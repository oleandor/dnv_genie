Command: CSRTankRun

Description:
CSR Tank
Example of use:
Define global and general code checking factors
Cc1.AddRun(CSRTankRun());
Cc1.run(1).generalOptions.parallelMode = true;
This command is scripted. The checks can be combined yield and buckling or separate buckling and yield.
Define global and panel specific code checking factors
Cc1.AddRun(CSRTankRun());
Cc1.run(1).panelOptions.unstiffened.idealizedLength = 1;
This command is scripted. Note that CSR Tank categorizes the panels into two groups,
stiffened or unstiffened. You can also force panels to be treated as stiffened or unstiffened.
Additional Information
Define global and general code, specific member and joint tubular
checking factors according to CSR Tank
