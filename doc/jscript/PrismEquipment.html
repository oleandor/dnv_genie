Command: PrismEquipment

Description:
Define a Prism Equipment
Example of use:
Define prismatic equipment for use in load cases to generate masses or loads
Equipment1 = PrismEquipment(5m,3m,2m,1000 tonne);
This command is scripted. Typically with height 2m, length 5m, width 3m and mass 1000 tonne
