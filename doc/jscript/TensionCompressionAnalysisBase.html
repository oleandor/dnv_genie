includeBeamElementForcesAndMoments

bool includeBeamElementForcesAndMoments()

Description:
*Include beam element forces and moments in result file* Specifies whether beam element forces and moments are stored on the Results Interface File. In order to perform code checks for beam elements, this option must be checked. Unchecking this option will reduce the size of the result file and reduce computational time.

includeLoadsFromInterfaceFiles

bool includeLoadsFromInterfaceFiles()

Description:
*Include loads from interface files in result file* Specifies that loads are transferred from the Input Interface File and Loads Interface File to the Results Interface File.Inertia loads specified as nodal accelerations (BNACCLO), or inertia loads (BGRAV or BRIGAC) are stored on the Results Interface File as equivalent distributed loads (BELOAD1) for all two-noded beam elements. In order to perform accurate code checks for beam elements with local loads and shell elements with local pressure, this option must be checked. Unchecking this option will reduce the size of the result file and reduce computational time.

includeShellElementStresses

bool includeShellElementStresses()

Description:
*Include shell element stresses in result file* Specifies whether shell element stresses are stored on the Results Interface File. Element stresses are also stored for truss elements. In order to perform code checks for shell elements, this option must be checked. Unchecking this option will reduce the size of the result file and reduce computational time.
