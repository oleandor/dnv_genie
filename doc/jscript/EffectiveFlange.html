Command: EffectiveFlange

Description:
Represent effective flange with. Use this property to specify the effective
flange width for stiffeners connected to plates
Example of use:
Create and edit an effective flange property
my_eff_flange = EffectiveFlange(0.8);
This command is scripted. Typically for an effective flange of 0,8 m
