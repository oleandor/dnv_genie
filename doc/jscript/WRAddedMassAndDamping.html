Command: WRAddedMassAndDamping

Description:
Added mass and damping only
Example of use:
Create Analysis with Deterministic seastates
Analysis1.step(2).addedMassAndDamping().calculateAddedMass(true);
Analysis1.step(2).addedMassAndDamping().calculateDamping(true);
Analysis1.step(2).addedMassAndDamping().includeInternalWater(true);
Analysis1.step(2).addedMassAndDamping().useCm(true);
This command is scripted. Added mass and damping are calculated and written to the
Loads Interface File independently of the choice of load calculation type.
The default value is not to calculate added mass and damping for transfer to
subsequent dynamic structural analysis. Please read the user manual GeniE Vol. 6
for more information about Added mass and damping.
