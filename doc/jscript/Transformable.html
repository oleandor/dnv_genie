copy3Point

ModelObject copy3Point(Point sp1, Point sp2, Point sp3, Point dp1, Point dp2, Point dp3)

Description:
Copy the object without scaling from one location to another.Copy the object without scaling from one location to another. The transformation is defined with 3 source and 3 destination positions. The 3 first define the source xy-plane, the 3 next define the destination xy-plane. Destination position dp1 will be equivalent to source position sp1. The direction from dp1 to dp2 will be equivalent to the direction from sp1 to sp2. The destination position triangle (dp1,dp2,dp3) need not be of the same shape as the source position triangle (sp1,sp2,sp3).
Parameters:
sp1 - Source: Anchor point
sp2 - Source: Point on x-axis
sp3 - Source: Point indicating y-axis
dp1 - Destination: Anchor point
dp2 - Destination: Point on x-axis
dp3 - Destination: Point indicating y-axis
Returns:

copyRotate

Array copyRotate(Point p1, LengthVector3d p2, Angle angle, int n)

Description:
Make n copies of the object,incrementing the angle for each copyMaken copies of the object. The transformation is defined as counterclockwise rotation around the given axis vector at the anchor point. The input angle defines the rotation for the first copy. For the subsequent copies, the actual rotation angle will be accumulated: iCopy * angle.
Parameters:
p1 - Point on rotation axis
p2 - Rotation axis vector
angle - Rotation angle
n - Number of copies
Returns:
