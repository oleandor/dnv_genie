Command: MorisonCoefficients

Description:
Represent Morison drag and inertia coefficients
Example of use:
Define constant Morison coefficients
MyMorison = MorisonCoefficients(0,0.7,0.7,0,2,2);
This command is scripted. For more details, please see
User Manual Volume 2 and the Wajac User Manual
