Command: GeometryRule

Description:
Control smartness of guide geometric creation. Specify whether GeniE shall
make groups when e.g. copying or moving structural members. This can lead to a
better performance when reading a command file or during modelling.
The command must be used with care.
Example of use:
Specify whether GeniE shall make groups when e.g. copying or moving structural members
GenieRules.Geometry.creationGrouping = cgGroupingJournal;
This command is scripted. Typically, when enabling grouping when reading a command file
