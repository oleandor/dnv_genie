Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: ImportSacsWind

Description:
Import a SACS input file. The stand-alone application SacsWind generates
the wind equipment needed by GeniE, storing them in SacsWindToGeniE.js.
Example of use:
Import a SACS input file
GeniE restores the information from the SACS input file by reproducing
the GeniE js-commands of relevant data.
