Command: ConductorShielding

Description:
Indicate that a plate is not watertight.
Example of use:
Define a plate or shell as permeable
MyPermeable = Permeable(true);
This command is scripted. Typically used when defining compartments.
Loads are applied to the plate or shell. Use this property on plates that
you want to be ignored when generating compartments
