Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: Water

Description:
Define water properties that can be used when defining a condition.
Example of use:
Water Categories
For defining a New Current Profile, go to:
          CurrentProfileRelDir

For defining a New frequency set, go to:
          FrequencySet

For defining a New Phase Set, go to:
          PhaseSet

For defining a New Wave Height Function, go to:
          WaveHeightFunction

For defining a New Wave Height Surface, go to:
          WaveHeightSurface

For defining a New Wave Height Set, go to:
          WaveHeightSet

For defining a New Regular Wave Set, go to:
          RegularWaveSet
The water properties may also be defined from the input dialogue for a
condition - the water properties will be found under the subfolder Water.
