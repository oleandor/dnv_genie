Command: MaterialLinear

Description:
Linear isotropic material
Example of use:
Create/Edit MaterialLinear
MyMat = MaterialLinear(275000, 7.85, 210000000, 0.3, 1.2e-5, 3.0e-5, 430000);
This command is scripted. Typically for a linear isotropic material named MyMat
