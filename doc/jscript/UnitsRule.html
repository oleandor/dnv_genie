Command: UnitsRule

Description:
Specify default input units and how labels etc, will appear on the desktop.
Example of use:
New workspace
GenieRules.Units.setOutputUnits("mm", "N", "delC");
GenieRules.Units.setInputUnit(Length, "mm");
GenieRules.Units.setInputUnit(Force, "N");
GenieRules.Units.setInputUnit(TempDiff, "delC");
GenieRules.Tolerances.useTolerantModelling = true;
GenieRules.Compatibility.version = "V4.0-14";
This command is not scripted. If non-default output units are selected
these will be logged. Typically when selecting "mm" as output units
