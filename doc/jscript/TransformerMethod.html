Command: TransformerMethod

 
Property Summary
 TransformerMethod tmUseDefaultTransformer read only
Copies the objects according to the old rule Individual Objects
 TransformerMethod tmUseIndividualObjectTransformer read only
Copies each object individually. For complex models, this may result in slow performance and ACIS errors.
 TransformerMethod tmUseModelTransformer read only
Copies the selected objects as a group. This is faster, but may lead to problems when joining the copied group with the rest of the model.
