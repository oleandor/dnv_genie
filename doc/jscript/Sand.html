Command: Sand

Description:
Represent a soil of type sand. The default values for frictionRatio,
zDisplacement and roWater may be overrided.
Example of use:
New Sand Profile
Sand1.ocRatio = 1;
Sand1.frictionRatio = 1;
Sand1.roTotal = 1.99 tonne/m^3;
Sand1.phi = 40 deg;
Sand1.zDisplacement = 0.99 m;
Sand1.initialModulus = 0.0 kN/m^3;
This command is scripted. Typically angle 40 deg, mass 1.99 tonne/m3,
over-consolidation ratio 1, friction ratio 1 and t-z curve z displacement 1m
