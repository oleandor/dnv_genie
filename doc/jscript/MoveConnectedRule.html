Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: MoveConnectedRule

Description:
Control connected move
Example of use:
Rules for connected move of beams
GenieRules.ConnectedMove.defaultConnected = true;
GenieRules.ConnectedMove.useStructuralPoints = true;
This command is scripted. Typically, when setting connected move as default
and including all structural points in calculation
 
Property Summary
 bool defaultConnected read/write
When true, autogeneration will be on by default in dialogs (GUI option only)
 bool rearrangeXJoints read/write
When true a connect move can rearrange an X-joints
 bool useStructuralPoints read/write
Specifies whether all structural points should be considered connected
