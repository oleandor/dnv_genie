Command: LayerSubdivisionOption

 
Property Summary
 LayerSubdivisionOption asIgnoreSubdivision read only
 LayerSubdivisionOption asMakeNewLayers read only
The 'Make new soil layers' option will create as many new soil layers as the other option creates sublayers. This option is believed to be better when the soil characteristics vary linearly within soil layers.
 LayerSubdivisionOption asSubdivideLayers read only
The 'Subdivide soil layers' option will change the number of sublayers within each layer. See the 'Sublayers' column in the Soil tab of the Location dialog. This option should be used when the soil characteristics are constant within soil layers. Note that often the default value of '1' make problems for Splice and in such cases this option may solve this problem.
