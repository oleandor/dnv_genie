Command: ExportResultsSin

Description:
Export FEA results as SIN file
Example of use:
Export a SIN results file with a user given name
SinExporter = ExportResultsSin();
SinExporter.DoExport("C:/Location/R1.SIN");
This command is not scripted. You can include the export capability by editing
the journal file to typically include the code above
To include SIN export in a journal file, see the last Chapter of User Manual Volume 6
