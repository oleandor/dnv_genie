Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: ChapterLoads

Description:
Document load parts. For more information, please read to GeniE Volume 6
Example of use:
ChapterMasses
a1 = Report("a");
a1.add(ChapterLoads());
a1.element(1).setLoadCases(Analysis1, Array(LC1_eqpm));
a1.element(1).setLoopLoads(true);
a1.element(1).add(TableLoadSummary());
a1.saveAs("a1.doc", mrWordXML);
This command is scripted. Typically when documenting the "Load Summary",
the "FEM Node Reaction" and the "FEM Node Displacement" for Analysis1
