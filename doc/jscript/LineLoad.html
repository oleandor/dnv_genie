Command: LineLoad

Description:
Define a LineLoad
Example of use:
Define a constant or linearly varying line load between two points along a line.
LLoad1 = LineLoad(LC1, FootprintLine(Point(4 m,0 m,10 m), Point(12 m,0 m,10 m)),
Component1dLinear(Vector3d(0 kN/m, 0 kN/m, -300 kN/m), Vector3d(0 kN/m, 0 kN/m, -100 kN/m)));
This command is scripted. Typically for a linear variation in z-direction
Define a constant or linearly varying line load between two points along a beam.
LLoad2 = LineLoad(LC1, FootprintBeam(Bm14, 0.2, 0.7),
Component1dLinear(Vector3d(0 kN/m, 0 kN/m, -300 kN/m), Vector3d(0 kN/m, 0 kN/m, -100 kN/m)));
This command is scripted. Typically for a constant line load in z-direction between relative positions 0.2 and 0.7 on beam Bm14
Define constant or linearly varying temperature intensity between two points along a beam
LLoad5 = LineLoad(LC1, FootprintLine(Point(2 m,0 m,1 m), Point(2 m,5 m,1 m)), Temperature1dLinear(100 delC, 100 delC));
This command is scripted. Typically for a constant temperature of 100 delC between two points
Notice that the temperature is constant over the beams cross section. It is possible to specify the temperature by referring
to two explicit points; in this case the load becomes independent of the beam and will not follow when the beam is moved or copied
There are two options:
Linear (as shown over or Javascript). Slide the "intensities" bar to change between the two options.
Specify the temperature by referring to relative positions of a beam(s)
LLoad6 = LineLoad(LC1, FootprintBeam(Bm20), Temperature1dLinear(50 delC, 100 delC));
This command is scripted. Typically for a varying temperature load between the two end positions on beam Bm20
Notice that the load becomes dependent of the beam and will follow when the beam is moved but not when it is copied.
There are two options:
Linear (as shown over or Javascript). Slide the "intensities" bar to change between the two options.
Create Line Component Load
return Vector3d(1N/m,x*2N/m2,0);
- input x,y,z, all of type Length
- output type Vector3d(ForcePerLength,ForcePerLength,ForcePerLength)
Create Line Component Load - Parametric
return Vector3d(1N/m,param*2N/m,0);
- input t, of type Number (unitless)
- output type Vector3d(ForcePerLength,ForcePerLength,ForcePerLength)
Create Line Temperature Load
return 1delC*y.toDouble();
- input x,y,z, all of type Length
- output TempDiff
Create Line Temperature Load - Parametric
return param*10delC;
- input t, of type Number (unitless)
- output type TempDiff
