Command: HydroBuoyancyArea

Description:
Buoyancy area for selected beam elements.
Example of use:
Define a buoyancy area
MyBuoancyArea= HydroBuoyancyArea(0.75m^2,0.15m^2);
This command is scripted. For more details, please see
User Manual Volume 2 and the Wajac User Manual
