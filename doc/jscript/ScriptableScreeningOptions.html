Screening criteria.
 
Property Summary
 bool CornerAngleLargerThan read/write
 Angle CornerAngleLargerThanValue read/write
Reports all elements with a corner angle larger than the threshold.
 bool CornerAngleSmallerThan read/write
 Angle CornerAngleSmallerThanValue read/write
Reports all elements with a corner angle smaller than the threshold.
 bool IsCollapsedElement read/write
Reports all collapsed elements. A collapsed element is a triangle or quadrilateral element of a zero area. A partially collapsed element is a quadrilateral element with overlapping node.
 bool IsDisconnectedElement read/write
Reports all elements disconnected from the model. Elements connected only to masses, dampers, springs are disconnected.
 bool IsDisconnectedNode read/write
Reports all nodes not connected to any element. Nodes only connected to linear couplings (support rigid links) are reported.
 bool IsOverlappingElement read/write
Reports all elements that overlap any other element. This check excludes beams.
 bool IsSestraQuadCriterionFailing read/write
Reports all elements that fail Sestra quad criterion. Sestra will give an error for a quad element if L1 > 25*L2, where L1 is length oflongest diagonal and L2 is length of shortest diagonal.
 bool IsSestraTriCriterionFailing read/write
Reports all elements that fail Sestra tri criterion. Sestra will give error for a triangle element if area < maxlen*maxlen/100 (where maxlen is length of longest side, area is triangle area).
 bool IsZeroPropertyElement read/write
Reports all elements without any properties. Elements without properties cannot be used in any analysis.
 bool MinimumJacobiSmallerThan read/write
 double MinimumJacobiSmallerThanValue read/write
Reports all elements with the Normalized Minimum Jacobian smaller than the threshold.
 bool OverlappingNodeWithinTolerance read/write
 Length OverlappingNodeWithinToleranceValue read/write
Reports all nodes which distance from other nodes is within the threshold.
 bool RelativeJacobiLargerThan read/write
 double RelativeJacobiLargerThanValue read/write
Reports all elements with the Relative Jacobi larger than the threshold.
 bool ShortEdgeLengthSmallerThan read/write
 Length ShortEdgeLengthSmallerThanValue read/write
Reports all elements with the shortest edge shorter than the threshold.
 bool WarpingRelDistanceLargerThan read/write
 double WarpingRelDistanceLargerThanValue read/write
Reports all elements with relative warping deviation larger than the threshold.
