Command: Clay

Description:
Represent a soil of type Clay. The default values for apiJFactor,
frictionRatio, zDisplacement and roWater may be overrided.
Example of use:
New Clay
Clay3.ocRatio = 1.0;
Clay3.frictionRatio = 1.0;
Clay3.zDisplacement = 1.0 m;
Clay3.roTotal = 1.94 tonne/m^3;
Clay3.apiJFactor = 0.5;
Clay3.epsc = 0.01;
Clay3.shearStrengthValues(200 KPa,0 m,100 KPa,-100 m);
This command is scripted. Typically with relevant data as shown
