Command: WRDeterministicSeastates

Description:
Define Deterministic Seastates
Example of use:
Create Analysis with Deterministic seastates
Analysis1.step(2).deterministicSeastates().populate();
Analysis1.step(2).deterministicSeastates().seastate(1).dataPhase(NoStretching,10 deg,36,true,NoDesignLoads,1,1);
Analysis1.step(2).deterministicSeastates().seastate(2).dataPhase(NoStretching,10 deg,36,true,NoDesignLoads,1,1);
Analysis1.step(2).deterministicSeastates().seastate(3).dataPhase(NoStretching,10 deg,36,true,NoDesignLoads,1,1);
Analysis1.step(2).deterministicSeastates().seastate(4).dataPhase(NoStretching,10 deg,36,true,NoDesignLoads,1,1);
Analysis1.step(2).deterministicSeastates().seastate(5).dataPhase(NoStretching,0,1,true, NoDesignLoads,1,1);
This command is scripted. Phase step, Time step or Single step can be selected.
Please notice that if you want to create analysis results for use in a gust wind analysis
in Framework only the option Single step can be used. Please read the user manual GeniE Vol. 6
for more information about Deterministic Seastates
Create Analysis with Deterministic seastates and water levels defined relative to seabed Z
Analysis1.step(2).deterministicSeastates().populate();
Analysis1.step(2).deterministicSeastates().seastate(1).dataPhase(WheelerStretching,5,72,buOn,MaxBaseShear,0.9,1,76.0,doOff);
Analysis1.step(2).deterministicSeastates().seastate(2).dataPhase(WheelerStretching,5,72,buOn,MaxBaseShear,0.9,1,75.5,doOff);
Analysis1.step(2).deterministicSeastates().seastate(3).dataPhase(WheelerStretching,5,72,buOn,MaxBaseShear,0.9,1,75.0,doOff);
Analysis1.step(2).deterministicSeastates().seastate(4).dataPhase(WheelerStretching,5,72,buOn,MaxBaseShear,0.9,1,74.5,doOff);
Analysis1.step(2).deterministicSeastates().seastate(5).dataPhase(WheelerStretching,5,72,buOn,MaxBaseShear,0.9,1,74.0,doOff);
This command is scripted. Phase step, Time step or Single step can be selected.
Please notice that if you want to create analysis results for use in a gust wind analysis
in Framework only the option Single step can be used. Please read the user manual GeniE Vol. 6
for more information about Deterministic Seastates
