Command: Shell

Description:
Define a Curved Shell
Example of use:
Skin/Loft Curves
Pl4 = SkinCurves(Array(Curve2, Curve3, Curve4));
Pl6 = LoftCurves(Pl2, Array(Curve2, Curve3, Curve4));
Code for Pl4: Similarly, when using plate Pl2 as guidance to control the initial surface of Pl4 (this is a lofting operation)
Code for Pl6: This command is scripted. Typically, when defining Pl4 from a skin operation between Curve1, Curve2 and Curve3
Curve-Net Interpolation
Pl5 = InterpolateCurveNet(Array(Curve1, Curve2, Curve3), Array(Curve4, Curve5));
Pl7 = InterpolateCurveNet(Array(Point1,Curve1, Curve2, Curve3), Array(Curve4, Curve5))
Code for Pl5: This command is scripted. Typically, when defining P15 from such an operation between
the curve arrays Array(Curve1, Curve2, Curve3) and Array(Curve4, Curve5)
Code for Pl7: Note that one of the curves may be a single point, provided that the same intersection restrictions hold for this point as well. For instance the command
will give a valid result, provided that Point1 is the intersection point Curve4 and Curve5, thus forming a triangular patch (which is a degenerate rectangular patch).
Sweep Curve
Pl9 = SweepCurve(Curve2, Curve5);
This command is scripted. Typically, when defining Pl9 by referring to Curve2 and Curve 5

Constructor Detail

CoverCurves

CoverCurves([ ...])

Description:
Generic cover curves functionality.CoverCurves has two modi:
- Cover planar curve: Create a complex set of curves making up a closed loop. CoverCurves will create a surface that fits the loop
- Skinning with guide curves: Create a set of skinning curves (may be of arbitrary complexity, but G1. Create simple Guide Curves that pass through all the skinning curves (defines the internal topology + bounding curve)
The skinning curves may not intersect.
The simple Guide Curves may not intersect.

The curves are analyzed internally, so the order which they are passed to CoverCurves is arbitrary.
