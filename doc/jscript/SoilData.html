Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: SoilData

Description:
Represent soil data of type skin friction or tip resistance.
Remember to add peakTipStress if tip resistance property.
Example of use:
New Soil Data
SoilData1.peakSkinFrictionTension = 3 KPa;
SoilData1.peakSkinFrictionCompression = 5 KPa;
SoilData1.soilShearModulus = -1 KPa;
SoilData1.poissonRatio = 0.5;
SoilData1.peakTipStress = 0 KPa;
SoilData1.peakTipStressDisplDiamRatio = 0.05;
SoilData1.peakSkinFrictionDisplDiamRatio = 0.01;
This command is scripted.

SoilData

Description:
This command creates soil data of type skin friction and/or tip resistance with constant peak skin friction
Parameters:
soilShearModulus - Initial value of soil shear modulus
poissonRatio - Soil Poisson ratio
peakSkinFrictionCompression - Peak skin friction in compression
peakSkinFrictionTension - Peak skin friction in tension
peakSkinFrictionDisplDiamRatio - Ratio between displacement to reach peak skin stress and pile diameter
peakTipStress - Peak tip stress acting at pile tip (equals 0.0 except when tip resistance property)
peakTipStressDisplDiamRatio - Ratio between displacement to reach peak tip stress and pile diameter
