Command: MorisonGlobalDirection

Description:
Define set of Morison coefficients depending on wave direction and seastate index
Example of use:
Define set of Morison coefficients depending on wave direction and seastate
MyMorisonGlobalDirection = MorisonGlobalDirection(Array(0,0),Array(0.7,0.9),Array(0.7,0.9),Array(0,0),Array(2,2.5),Array(2,4),Array(0 deg,0 deg), Array(1,4));
This command is scripted. In this case it consists of coefficients for two directions/seastates.
For more details, please see User Manual Volume 2 and the Wajac User Manual
