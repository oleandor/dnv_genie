Command: BeamJoiner

Description:
Beam joining properties in Fem import
 
Property Summary
 Angle AngularTolerance write only
Connected beam elements are joined only if the angle between them along beam is within this tolerance.
 bool CompareMaterials write only
Connected beam elements are joined only if they have same material properties.
 bool CompareSections write only
Connected beam elements are joined only if they have same section properties.
 bool CompareSets write only
Connected beam elements are joined only if they are in same set.
 Angle DirectionTolerance write only
Connected beam elements are joined only if their direction vectors are within this tolerance.
 Length EccentricityTolerance write only
Connected beam elements are joined only if their eccentricity properties are within this tolerance.
 bool JoinBeamsWithOppositeDirection write only
This option allows the beams to be joined if two beam elements have the same local Z, but are modeled in opposite directions It is recommended to use this setting when importing ship models.
