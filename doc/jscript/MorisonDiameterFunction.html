Command: MorisonDiameterFunction

Description:
Define Morison drag and inertia coefficients as a function of element diameter, values may be different for fouled and non-fouled members
Example of use:
Define Morison coefficients as a function of diameter
MyMorisonDiameter = MorisonDiameterFunction(Array(2,3), Array(1,0.9),Array(2,2.1));
This command is scripted. Typically (for 2 different diameters).
For more details, please see User Manual Volume 2 and the Wajac User Manual
