Command: TolerancesRule

Description:
Control the tolerances used in tolerant modelling
Example of use:
Set tolerances when working with tolerant modelling
GenieRules.Tolerances.pointTolerance = 0.05;
GenieRules.Tolerances.angleTolerance = 3 deg;
Sp6.boundary = BoundaryCondition(Dependent, Dependent, Dependent, Dependent, Dependent, Dependent);
This command is scripted. Typically, when modifying the point tolerance to 0.05 m and angular tolerance to 3 degrees
