Command: GuideEllipse

Description:
Represent an ellipse or elliptic arc guiding curve
Example of use:
Insert a guide arc elliptic graphically by snapping to 3 points (origin, start and end of ellipse)
Curve20 = GuideArcElliptic(Point(10 m,2.5 m,7.5 m), Point(10 m,5 m,5 m), Point(10 m,2.5 m,10 m), true);
This command is scripted.
Insert a guide circle graphically by snapping to 3 points (origin, radius, plane)
Curve24 = GuideCircle(Point(10 m,2.5 m,2.5 m), Point(10 m,5 m,2.5 m), Point(10 m,0 m,5 m));
This command is scripted.
