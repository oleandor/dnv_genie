Command: WRSpecialOptions

Description:
Eliminate structural parts from the wave load analysis.
You can also add additional still water levels
Example of use:
Eliminate, Exclude and Add specialOptions
Analysis1.step(2).specialOptions().excludeEccentricities(true);
Analysis1.step(2).specialOptions().useEliminatedStructure(true);
Analysis1.step(2).specialOptions().eliminatedStructure = Topside;
Analysis1.step(2).specialOptions().absoluteWaterDepths(false);
Analysis1.step(2).specialOptions().removeAllLevels();
Analysis1.step(2).specialOptions().addLevel(2m);
Analysis1.step(2).specialOptions().addLevel(-3m);
This command is scripted. By excluding eccentricities, eliminating the named set "Topside"
and adding two new still water levels at +2m and -3 m relative to the base still water level.
Note that this is the default behavior. Thus, the "absoluteWaterDepths(false)" command is not needed.
Please read the user manual GeniE Vol. 6 for more information about additional details for SpecialOptions
Added Mass and Damping only
Analysis1.step(2).addedMassAndDampingOnly(true);
Analysis1.step(2).addedMassAndDamping().calculateDamping(true);
Analysis1.step(2).addedMassAndDamping().useCm(true);
Analysis1.step(2).addedMassAndDamping().includeInternalWater(true);
This command is scripted. This option implies that no load cases will
be calculated and written to the loads interface file – only added mass and damping are considered.
Please read the user manual GeniE Vol. 6 for more information about additional details about Added Mass and Damping
Data Check Only
Analysis1.step(2).dataCheckOnly(true);
This command is scripted. The data check option prevents Wajac from a full execution of the analysis;
a data check will be performed of the input data to determine whether they are consistent.
Please read the user manual GeniE Vol. 6 for more information about additional details about Data Check Only
Automatic generation of input files
Analysis4.step(2).generateInput = false;
This command is scripted. The default option (“True”) implies that the input file for Wajac is automatically generated.
In other words, if you modify one of the parameters this file is automatically updated prior to the next analysis run.
If you want to have a manual control of the input file you should deselect this option.
Please read the user manual GeniE Vol. 6 for more information about additional details about generation of input files
RelativeAdditionalDepths and Add specialOptions
Analysis1.step(2).specialOptions().absoluteWaterDepths(true);
Analysis1.step(2).specialOptions().removeAllLevels();
Analysis1.step(2).specialOptions().addLevel(52m);
Analysis1.step(2).specialOptions().addLevel(47m);
This command is scripted. Two new still water levels are added at 52m and 47 m relative to the seabed Z.
Note that the default behavior is that additional water levels are defined relative to Location water level Z.
Thus, the "absoluteWaterDepths(true)" command is needed.
Please read the user manual GeniE Vol. 6 for more information about additional details for SpecialOptions
