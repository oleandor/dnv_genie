Command: ImportGensod

Description:
Import a GENSOD input file. The stand-alone application GENSOD generates
the soil data needed by SPLICE, storing them in GENSOD.inp.
Example of use:
Import a GENSOD.inp file
GeniE restores the information from the GENSOD.inp file by reproducing
the js-commands, thus supplying the folders "Environment/Location" and
"Environment/Soil" of the GeniE browser with the relevant data.
