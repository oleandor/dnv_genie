Command: ExportGeometryRhino

Description:
Export guiding geometry to a file which can be read by the Rhino-plug in GenieRhinoConverter
Example of use:
Export all guiding curves
GGExporter = ExportGeometryRhino();
GGExporter.DoExport("C:/Location/gg_curve.grc");
