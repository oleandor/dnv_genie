Command: Reinforcement

Description:
Represent a reinforcement (Can/Stub)
Example of use:
Create a reinforcement to be applied to a can or stub
MyReinforcement = Reinforcement(0.5,0.45m);
This command is scripted.
Additional Information
Use this property on segments that you want to be threat as cans/stubs.
The length of this reinforcement is calculated by the formula: 'length=max(fac*diameter,min);'
