Command: EquipmentConfig

Description:
Equipment Configuration
Example of use:
Define the COG manually relative to the footprint centre
Equipment1.centreOfGravity (Vector3d(1m,1m,3m));
This command is scripted. Typically when defining the COG position at (1m, 1m, 3m)
Define the footprint of the equipment
Equipment1.clearFootprint();
Equipment1.addToFootprint(1.25 m, 2.5 m, 0.75 m, 1.5 m);
Equipment1.addToFootprint(1.25 m, 2.5 m, -1.5 m, -0.75 m);
Equipment1.addToFootprint(-2.5 m, -1.25 m, 0.75 m, 1.5 m);
Equipment1.addToFootprint(-2.5 m, -1.25 m, -1.5 m, -0.75 m);
This command is scripted. There are 4 templates that can be edited.
The footprint may be outside the equipment border and dependent on which
template is chosen and the size of each footprint
Specify whether generated loads shall be constant or linear
LC_heli.setCurrent();
LC_heli.constantLoad(Helideck);
LC_heli.varyingLoad(Helideck);
This command is scripted. Typically when altering to constant loads or linear
for the equipment Helideck in load case LC_heli
Specify the exact location of the equipment
LC_heli.placeAtPoint(Helideck, Point(15m,20m,12m), LocalSystem(Vector3d(1 m,0 m,0 m), Vector3d(0 m,0 m,1 m)));
This command is scripted. Typically when modifying the location to (15m, 20m, 12m)
