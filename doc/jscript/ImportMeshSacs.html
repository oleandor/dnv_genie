Command: ImportMeshSacs

Description:
Import Finite Element information from SACS. The items that are
converted are documented on the Reference Document section on GeniE's help pages.
Example of use:
Import geometry and loads from a Sacs input file and create concept model
SacsImporter = ImportMeshSacs();
SacsImporter.setOutputUnits("m","N");
SacsImporter.DoImport("C:/Location/yourname.inp");
This command is scripted and a typical example is given above
