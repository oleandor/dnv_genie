Command: HoleMeshingOptions

 
Property Summary
 HoleMeshingOptions hmIgnoreHole read only
Holes with this option will be ignored during meshing. No trace of them will remain on the mesh.
 HoleMeshingOptions hmMeshHole read only
Holes with this option will be meshed. The number of layers chosen ensures at least that number of perfect offset layers around the hole.
 HoleMeshingOptions hmRemoveElementsByAreaFraction read only
Holes with this option will be ignored during meshing. After mesh is complete, elements whose intersection with the hole has area fraction larger than threshold value will be removed automatically.
 HoleMeshingOptions hmRemoveElementsFromInsideHole read only
Holes with this option will be ignored during meshing. After mesh is complete, elements with centroid inside the hole will be removed automatically.
