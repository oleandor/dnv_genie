Command: ImportMeshStrucad3d

Description:
Import Finite Element information from Strucad3d The items that are converted are
documented on the Reference Document section on GeniE's help pages.
Example of use:
Import geometry and loads from a Strucad input file and create concept model
Strucad3dImporter = ImportMeshStrucad3d();
Strucad3dImporter.setOutputUnits("m","N");
Strucad3dImporter.DoImport("C:/Location/yourname.s3d");
This command is scripted and a typical example is given above
