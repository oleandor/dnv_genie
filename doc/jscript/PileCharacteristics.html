Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: PileCharacteristics

Description:
Define specific pile properties to be used in a non-linear pile soil analysis.
Example of use:
Define specific pile properties
MyPileChar = PileCharacteristics(2.5, tcFree);
This command is scripted.
