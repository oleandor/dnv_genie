Command: LinearAnalysis

Description:
Define a Linear Analysis
Example of use:
Execute a linear structural analysis with or without wave loads, pile and soil
Analysis1 = Analysis(true);
Analysis1.add(MeshActivity());
Analysis1.add(LinearAnalysis());
Analysis1.add(LoadResultsActivity());
Analysis1.execute();
This command is scripted. The analysis programs Sestra, Wajac and Splice are used by GeniE to
solve the analyses; these programs require a separate license key to run
The Mesh Subset option
Analysis1.step(1).subset = bottom_deck;
Analysis1.step(1).meshPriority = MeshPriority1;
This command is scripted. The Mesh Subset option is used to mesh parts of the concept model only.
A boundary condition may also be part of a set - this means that a complete finite element model may
be generated for sets (for analysis or for creation of super elements) as intersecting loads are automatically included
Datacheck Only
Analysis1.step(2).dataCheckOnly = true;
This command is scripted. The Datacheck Only will set up the input file to Sestra so a
data check is performed but no execution of finite element analysis.
Keep the input file after Analysis
Analysis1.step(2).generateInput = false;
This command is scripted. If you want to edit and keep the input file to Sestra you should deselect
the Automatic generation of input files. If you do not deselect this option the modified input file to
Sestra is not kept for subsequent analysis.
Option for Warp Correction
Analysis1.step(2).warpCorrection = false;
This command is scripted. The option for Warp Correction will use Sestra's option for improving
the stiffness for warped 4 noded shell elements (i.e. elements with "nodes out of plane").
Execute and control input data for an eigenvalue analysis - 1
Analysis1 = Analysis(true);
Analysis1.add(MeshActivity());
Analysis1.add(LinearAnalysis());
Analysis1.step(2).setEigenvalueAnalysis(egLanczos, 10);
Analysis1.add(LoadResultsActivity());
Analysis1.execute();
This command is scripted. The default option is to use Lanczos solver with 10 eigen-modes.
Execute and control input data for an eigenvalue analysis - 2
Analysis1.step(2).setEigenvalueAnalysis (egMultifrontLanczos, 15, 2);
This command is scripted. The following eigenvalue algorithms are available for free vibration analysis:
- An implicitly restarted Lanczos' method combined with a multifrontal solver
- Householder's method
- Subspace Iteration
- Lanczos' method
Option for Result File Format
Analysis1.step(2).resultFileFormat = SIN_Norsam;
This command is scripted. The following result file formats are available:
- Norsam(SIN)
- Formatted(SIF)
- Unformatted(SIU)
Execute and control input data for a dynamic analysis
Analysis1 = Analysis(true);
Analysis1.add(MeshActivity());
Analysis1.add(LinearAnalysis());
Analysis1.step(2).setDynamicAnalysis();
Analysis1.add(LoadResultsActivity());
Analysis1.execute();
This command is scripted.
