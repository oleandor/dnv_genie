Command: CorrosionAdditionZLevel

Description:
Define corrosion addition as a function of vertical position.
Example of use:
Define a absolute linearly varying value for corrosion addition with elevations relative to SWL
MyCorrosionAddition = CorrosionAdditionZLevelFunction();
MyCorrosionAddition.add(5m,0.02m);
MyCorrosionAddition.add(0m,0.03m);
MyCorrosionAddition.add(-50m,0.01m);
MyCorrosionAddition.setAbsoluteElevations(false);
This command is scripted
Define a constant value for corrosion addition with elevations relative to seabed Z
MyCorrosionAddition = CorrosionAdditionZLevelFunction();
MyCorrosionAddition.add(5m,10m,0.015m, prUpperZone);
MyCorrosionAddition.add(0m,5m,0.025m);
MyCorrosionAddition.add(-50m,0m,0.02m);
This command is scripted
