Command: ExportConceptXml

Description:
Export Conceptual information to Xml. The parts of the concept model
that are exported to xml format are documented in User Manual Volume 3.
Example of use:
Export concept model to XML forma
XmlExporter = ExportConceptXml();
XmlExporter.exportLoads = true;
XmlExporter.exportCodeChecks = true;
XmlExporter.DoExport("C:/Location/yourname.xml");
This command is not scripted. You can include the export capability by editing
the journal file to typically include the code above
To include XML export in a journal file, see the last Chapter of User Manual Volume 6
