Command: WROutput

Description:
Specify Output details in Analysis. Specify the overturning moment reference point,
how much to add to the printed file from Wajac and details for the loads interface file.
Example of use:
Specify the overturning moment reference point
Analysis1.step(2).output().globalResults().momentRefPoint = Point(0 m,0 m,50 m);
This command is scripted. By typically specifying the z-value to 50 meters
Please read the user manual GeniE Vol. 6 for more information about Output
Add additional details to the Wajac listing file
Analysis1.step(2).output().printFile().addDeterministicData(Jacket,2,10);
This command is scripted. By selecting the named set "Jacket", sea state number "2" and step "10"
The Print file option allows you to add additional details to the Wajac listing file from the hydrodynamic analysis.
Please read the user manual GeniE Vol. 6 for more information about additional details for Output
Control type of file format and the first load case number
Analysis1.step(2).output().loadsInterfaceFile().formatted(false);
Analysis1.step(2).output().loadsInterfaceFile().loadCaseNumbering(true);
Analysis1.step(2).output().loadsInterfaceFile().firstLoadCaseNumber = 45;
This command is scripted. By selecting unformatted files and start of first wave load case number to "45"
The Loads interface file option controls type of file format and a manual control of the first load case number
Please read the user manual GeniE Vol. 6 for more information about additional details for Output
