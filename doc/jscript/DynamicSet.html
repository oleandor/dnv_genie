Command: DynamicSet

Description:
Represent a set of modeller objects
Example of use:
Create a dynamic set
Pipe1 = PipeSection(1, 0.05);
ISect1 = ISection(0.8, 0.4, 0.02, 0.02);
Bm1 = StraightBeam(Point(0 m,0 m,0 m), Point(0 m,10 m,0 m));
Bm2 = StraightBeam(Point(10 m,0 m,0 m), Point(10 m,10 m,0 m));
Bm1.section = Pipe1;
Bm2.section = ISect1;
ds1 = DynamicSet(LimitBox(Point(-1, -1, -1), Point(1, 11, 1)));
// ds1 will now contain Bm1
ds2 = DynamicSet(LimitString("Section", "ISect1", true));
// ds2 will now contain Bm2
Bm1.section = ISect1;
// ds2 will now contain both Bm1 and Bm2
This command is scripted.
Move content from dynamic set to ordinary set
var arrayOfObjectsInSet = new Array();
for (var namedSetObject in ModelObjects){
if( namedSetObject.supportsType(typeSet) && (namedSetObject.name()=="DynSet") ) {
for (var objectInSet in namedSetObject){
arrayOfObjectsInSet[arrayOfObjectsInSet.length] = objectInSet;
} } }
MySet = Set();
for (i=0;i<=arrayOfObjectsInSet.length-1;i++){
objectToAdd = arrayOfObjectsInSet[i];
MySet.add(objectToAdd);
}
Delete(DynSet);
Moving everything from the dynamic set "DynSet" to the set "MySet", then deleting "DynSet"
