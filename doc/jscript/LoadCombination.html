Command: LoadCombination

Description:
Define a Load Combination
Example of use:
Define load combinations built up of other load cases
LC_Comb = LoadCombination();
LC_Comb.addCase(LC1, 1);
LC_Comb.addCase(LC3, 1);
This command is scripted.
It is possible to use a load combination in another load combination.
Define load combinations built up of other load cases, with one scanCase holding complex data
LC_Comb = LoadCombination();
LC_Comb.addCase(LC1, 1, false);
LC_Comb.addCase(LC2, 1, false);
LC_Comb.addCase(LC3, 1, true);
This command is scripted.
It is possible to use a load combination in another load combination.
Define load combinations built up of other load cases, with convertLoadToMass and globalScaleFactor
LC_Comb = LoadCombination();
LC_Comb.addCase(LC1, 1);
LC_Comb.addCase(LC3, 1);
LC_Comb.convertLoadToMass = true;
LC_Comb.globalScaleFactor = 0.75;
This command is scripted.
It is possible to use a load combination in another load combination. However, only top level load combination can have convertLoadToMass = true
