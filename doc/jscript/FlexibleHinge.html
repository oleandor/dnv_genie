Command: FlexibleHinge

Description:
Beam end hinge property. dx,dy,dz and rx,ry,rz are absolute stiffnesses.
A value of -1 corresponds to infinite stiffness.
Example of use:
Specify a flexible where stiffness for each degree of freedom is specified
MyFlexHinge = FlexibleHinge(Fixed,Fixed, Stiffness(500),Stiffness(12500), Stiffness(25000),Stiffness(75000));
This command is scripted. Typically for a flexible hinge where 4 different
spring stiffnesses are used (fixed in x and y)
The Extreme Values
The extreme values of a fixation is described by:
a=0, no stiffness
a=+inf, infinite stiffness
a=-1, infinite stiffness
