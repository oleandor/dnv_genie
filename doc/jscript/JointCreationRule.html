Command: JointCreationRule

Description:
This rule will only be used by the GUI to call the correct script commands
to generate new joints. Rules for automatic creation of joints.
The joints will be defined and named according to the rules specified as well as the naming convention.
Example of use:
Joint creation
GenieRules.JointCreation.exclude(geFreeBeamEnds,true);
GenieRules.JointCreation.exclude(ge2BeamAligned,true);
GenieRules.JointCreation.exclude(geThroughBeamPure,true);
GenieRules.JointCreation.autoGenerate = true;
This command is scripted. Typically, when activating the automatic joint generation and the exclusion criteria as shown

selectionAware

bool selectionAware()

Description:
Specifies whether the topology of the selection area is significant when creating joints in a selectionWhen selectionAware is false, the entire model topology will be taken into account when evaluating joint exclusion rules.
When selectionAware is true, only the topology of the selection will be taken into account evaluating joint exclusion rules.
