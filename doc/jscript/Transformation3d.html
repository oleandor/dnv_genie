Command: Transformation3d

Description:
Represent a 3d transformation matrix
Example of use:
Create a 3d transformation matrix
trans=Transformation3d();
trans.rotate(Point(5m,5m,0m),Vector3d(0,0,1),45deg);
trans.translate(Vector3d(1m,0m,0m));
Creates a unit transformation. Translate this rotation 1m along the positive x-axis
trans now represents a translation along positive x-axis
