Command: CutPlane

 
Function Summary
   clipAnnotations(bool)
If enabled, annotations will be tested against the plane when they are inserted into the scene. If the test fails they will not be inserted. This is distinct from cutting, which cuts objects already in the scene.
   setActive(bool)
If false the cut plane will not remove any geometry from display
   showGraphics(bool)
Show or hide the graphical representation of the cutplane e.g. a polygon
   width(double)
Internal function to store slice definition
