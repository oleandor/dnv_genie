Command: ImportRuleLoadsXml

Description:
Import Rule Loads as XML file. Import loads and boundary conditions as defined
by Nauticus Hull using the CSR rules for bulk ships.
Example of use:
Import geometry and create concept model
RuleLoadsImporter = ImportRuleLoadsXml();
RuleLoadsImporter.DoImport("C:/Location/ yourname.xml");
This command is scripted and a typical example is given above
