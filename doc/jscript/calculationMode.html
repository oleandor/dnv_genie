Overview Class Hierarchy  GeniE V8.5-04 25-Jan-2023

Command: calculationMode

 
Property Summary
 calculationMode cmAutomatic read only
Automatic: the program decides whether non-linear calculation is needed or not, and runs non-linear if needed, otherwise linear is run.
 calculationMode cmLinearOnly read only
Linear: runs a linear calculation, which gives no detailed results – only OK/NOT OK, and is very quick.
 calculationMode cmNonLinear read only
Non-linear: runs a full PULS non-linear check for all capacity models. This is the most time consuming which gives the most detailed results.
