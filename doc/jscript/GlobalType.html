mpMaxChord

MeshSettingType mpMaxChord()

Description:
Set a mesh density to curved edges based on the curvature. The value selected is the relative distance to the curve. The distance if from the middle of the straight element edge between the corner nodes to the curve, and the size is relative to the length of the same straight element edge.

mpMaxRelativeJacobi

MeshSettingType mpMaxRelativeJacobi()

Description:
Maximum Jacobi determinant of a node divided by the minimum The Jacobi determinant measures the quality of quad element. If the relative value is 1, the element is a perfect square. If the relative value is large the element may be seriously misshaped, and unacceptable to the analysis program.

mpMaxTwistAngle

MeshSettingType mpMaxTwistAngle()

Description:
Max twist angle of a quad element. The twist angle is the angle between the element normal in the different nodes. The normal for a node is calculated using the neighbour nodes to define a plane. The maximum of the 4 angles is tested.

mpMinAngle

MeshSettingType mpMinAngle()

Description:
Minimum angle of a quad element.

mpMinEdge

MeshSettingType mpMinEdge()

Description:
Eliminate small edges relative to mesh density Makes sure that small details in the geometry model do not lead to tiny element edges. The value is relative to the mesh density for the part. The options mpMinEdge and mpMinEdgeByLength are mutually exclusive.

mpMinEdgeByLength

MeshSettingType mpMinEdgeByLength()

Description:
Eliminate small edges by absolute length Makes sure that small details in the geometry model do not lead to tiny element edges. The value is in coordinate length. The options mpMinEdge and mpMinEdgeByLength are mutually exclusive.

mpMinMaxDensityRatio

MeshSettingType mpMinMaxDensityRatio()

Description:
Density on edges with no enforced density will be calculated from the minimum and maximum density specified on the other edges of a face The density will be set to maximum of minimum face density or mimMaxDensityRatito multiplied by maximum density. I.e. if ratio is 0, minimum density is used and if ratio is 1.0 maximum density will be used. Values outside the range 0-1 is not accepted. This feature is especially useful for areas of the model where the user does not want to specify a density.

mpMinNonConceptualEdge

MeshSettingType mpMinNonConceptualEdge()

Description:
Eliminate non-conceptual small edges by absolute or relative length Makes sure that small mesh edges generated due to modeling mismatches are collapsed as a post-meshing step.

mpMinNormalizedJacobi

MeshSettingType mpMinNormalizedJacobi()

Description:
Minimum normalized Jacobi determinant of the nodes of quad elements. The Jacobi determinant shall be > 0 for all nodes of an acceptable element. Small positive values indicates bad shape. The determinant is normalized to make the minimum value = 1 for the perfect square elements.
