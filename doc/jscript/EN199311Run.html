Command: EN199311Run

Description:
Eurocode3 EN 1993-1-1 2005
Example of use:
Define global and general code checking factors - Standard
Cc1.AddRun(EN199311Run());
Cc1.run(1).generalOptions.nationalAnnex = naStandard;
Cc1.run(1).generalOptions.partialFactorM0 = 1;
Cc1.run(1).generalOptions.partialFactorM1 = 1;
Cc1.run(1).generalOptions.method1 = true;
This command is scripted, for generalOptions.annex "Standard"
Define global and general code checking factors - Danish Normal
Cc1.AddRun(EN199311Run());
Cc1.run(1).generalOptions.nationalAnnex = DanishNormal;
Cc1.run(1).generalOptions.partialFactorM0 = 1.1;
Cc1.run(1).generalOptions.partialFactorM1 = 1.2;
Cc1.run(1).generalOptions.method1 = true;
This command is scripted, for generalOptions.annex "Danish Normal"

Cc1.run(1).generalOptions.partialFactorM0 = 1.05;
Cc1.run(1).generalOptions.partialFactorM1 = 1.05;
Cc1.run(1).generalOptions.method1 = true;
This command is scripted, for generalOptions.annex "Norwegian grouse"
Define global and member specific code checking factors
Cc1.run(1).memberOptions.LTBFactorC1 = 3;
This command is scripted, typically when modifying the lateral
torsional buckling Factor C1 to "3"
Additional Information
Define global and general code, specific member and joint tubular checking factors
according to Eurocode3 EN 1993-1-1 2005. There are
different national annexes; they have different safety factors.
