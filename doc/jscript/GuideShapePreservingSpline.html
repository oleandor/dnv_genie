GuideShapePreservingSpline

GuideShapePreservingSpline(Array points, bool is periodic, LengthVector3d v-start, LengthVector3d v-end, SplineType spline-type, SplineParameterizationType parameterization-type)

Description:
Create an open shape-preserving guide spline
Parameters:
points - Points defining the guide curve
is periodic - Boolean to distinguish between open and periodic curves
v-start - Vector defining the guide curve tangent-vector at the start point
v-end - Vector defining the guide curve tangent-vector at the end point
spline-type - Type of Variable Degree Spline
parameterization-type - Calculation of parametric values
