nameMask

  nameMask(string mask, bool isRegexp)

Description:
Accept only sections that matches the specified name mask
Name mask
  Wildcard: Use a '*' to select any character
  Regexp: The most usefull rules are:
          Use any character except:
                 '.', '|', '*', '?', '+', '(', ')', '{', '}', '[', ']', '^', '$' and '\'
                    represents themselves
                   To represent these, use a '\' in front of it
          Use a '.' to select any single character
                '*' to accept previous character or expression repeated zero or more times
                '+' to accept previous character or expression repeated one or more times
                '?' to accept previous character or expression repeated zero or one time
                '(' and ')' to create a subexpression
                '|' to accept both alternative subexpressions
                '[abf]' to accept the characters 'a', 'b' and 'f'
                '[a-f]' to accept the characters in the range from 'a', to 'f'
Parameters:
mask - Name mask in wildcard or regexp format. See detailed description
isRegexp - false: wildcard, true: regexp
