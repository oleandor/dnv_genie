Command: CapacityManager

Description:
A collection of capacity models and options for the Capacity Manager
Example of use:
Create a Capacity Manager
Cc1 = CapacityManager(Analysis1);
This command is scripted. Typically when using "Analysis1" as basis for the capacity manager.
Update Members from Structure
Cc1.updateMembersFromStructure();
This command is scripted. This is used when you change something in the concept model, like removing
a beam or altering segmentation, and you want this change to be reflected in the capacity model.
Information is moved from the concept model to the capacity model.
Update Structure from Member
Cc1.updateStructureFromMembers();
This command is scripted. This is used to move information from the capacity model to the concept model.
Set active
Cc1.setActive();
This command is scripted, typically when switching from capacity manager Cc2 to Cc1
Edit Description
Cc1.description = "UM example";
This command is scripted, typically when adding the text UM Example
Delete
Delete(Cc1);
This command is scripted, typically when deleting a capacity manager or a code check run
Rename
Rename(Cc1,"UM_Cap_Manager");
This command is scripted, typically when renaming to UM_Cap_Manager
Properties
This command is scripted. The scripting depends on which modifications are
performed to the capacity manager or a code check run.
