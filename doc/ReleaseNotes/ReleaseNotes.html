  • Feature 248370: GeniE export of pile/soil data to Usfos without running Wajac and Splice

  • Feature 271222: Support ISO 19902 edition 2020 Conical segment code check

  • Feature 335188: Include internal surfaces inside compartments when creating dummy hydro pressure

  • Feature 358423: Implement Genie Runtime - Genie without GUI

  • Feature 364200: Rule Load XML format extension for JS sea pressure and compartment loads

  • Feature 399478: Upload data to Sesam Insight from GeniE UI

  • Technical Feature 405663: Upgrade to Hoops v27.20
